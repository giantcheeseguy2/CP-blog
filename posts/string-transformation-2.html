Solution

We can represent every index as an edge from \(a[i]\) to \(b[i]\), thus turning our problem into a directed graph. This graph doesn't have much meaning yet, but lets try putting some conditions our graph to solve an easier version of the problem. Notice if our graph is a connected dag of size \(n\), then there is a solution that involves \(n - 1\) moves. We can't do better than this, since \(n - 1\) is the bare minimum amount of edges needed to keep the graph connected. To achieve the \(n - 1\), we simply take a topological sort of the dag, call it \(t\), and whenever we are at \(t_i\), we update all characters \(t_i\) to be \(t_{i + 1}\). It is easy to see that this will always work, since if there is an edge from \(a\) to \(b\) then \(a\) will come before \(b\) in \(t\). Thus, \(a\) will reach \(b\) after some series of operations. Now, consider if there are cycles in the graph. There is an underlying dag structure in this graph, so lets take the topological sort of that. Then, if a node has an edge going up the dag and creating a cycle, then we can additionally add that node to the front of the topological sort, thus satisfying all of that node's outgoing edge requirements. However, there are many underlying dag structures to choose from. We want to minimize the amount of nodes we must add to the front of the topological sort, so we want to minimize the amount of nodes that have an edge pointing back into the dag. This is equivalent to choosing a maximal subset of nodes such that no cycle of forms, which can be done using bitmask dp. Note that the outgoing edges from each node can be represented by a bitmask to improve complexity. Also note that the graph is not necessarily connected, so we have to account for each component as well.

Comments

void setIO() {
    ios_base::sync_with_stdio(0); cin.tie(0);
}

int par[20];

int find(int x){
    if(x == par[x]) return x;
    return par[x] = find(par[x]);
}

int main(){
    setIO();
    int T;
    cin >> T;
    for(int tt = 1; tt <= T; tt++){
        int n;
        cin >> n;
        string a, b;
        cin >> a >> b;
        for(int i = 0; i < 20; i++) par[i] = i;
        int g[20];
        memset(g, 0, sizeof(g));
        for(int i = 0; i < n; i++){
            par[find(a[i] - 'a')] = find(b[i] - 'a');
            if(a[i] != b[i]){
                g[a[i] - 'a'] |= 1 << (b[i] - 'a');
            }
        }
        set<int> comp;
        for(int i = 0; i < 20; i++) comp.insert(find(i));
        int mx = 0;
        bool dp[1 << 20]; 
        memset(dp, false, sizeof(dp));
        dp[0] = true;
        for(int i = 1; i < (1 << 20); i++){
            for(int j = 0; j < 20; j++){
                if(i >> j & 1){
                    dp[i] |= dp[i ^ (1 << j)] && ((g[j] & i) == 0);
                }
            }
            if(dp[i]) mx = max(mx, __builtin_popcount(i));
        }
        cout << 40 - mx - comp.size() << endl;
    }
}
