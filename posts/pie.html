Goal

We are given \(N\) possibly intersecting sets and we want to find the union of those sets. However, directly computing the union is difficult. Instead, given some amount of sets, we are able to find the intersection of those sets easily.

Solution

We can apply the principle of inclusion and exclusion to solve this problem. For a small case of two sets \(A\) and \(B\), the answer is just \(|A| + |B| - |A \cap B|\). For a case of three sets \(A\), \(B\), and \(C\), the answer is then \(|A| + |B| + |C| - |A \cap B| - |A \cap C| - |B \cap C| + |A \cap B \cap C|\). We can observe that whenever there is a combination of an odd number of sets, we want to add it. when the amount is even we want to subtract it. This is just removing the overcount and filling in the undercount. Formally, for a set \(S\), \(Union(S) = \sum_{T \subseteq S} (-1)^{|T| + 1} \cdot Intersection(T)\). We cn also use PIE to find the intersection of a set, \(Intersection(S) = \sum_{T \subseteq S} (-1)^{|T|} \cdot Union(T)\).

Further Applications

Lets say we have a set of elements \(a_i\). If you think about each \(a_i\) as a prefix on the number line \([1, a_i]\), then the union of two elements is \(Max(a_i, a_j)\), and the intersection is \(Min(a_i, a_j)\). Thus, \(Max(S) = \sum_{T \subseteq S} (-1)^{|T| + 1} \cdot Min(T)\). Similarly, \(Min(S) = \sum_{T \subseteq S} (-1)^{|T|} \cdot Max(T)\).
